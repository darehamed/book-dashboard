# Book Dashboard

Full-stack application for managing a book collection.

## Structure
- `/backend` - NestJS GraphQL API
- `/frontend` - React + Chakra UI frontend

## Local Development

### Backend
```bash
cd backend
npm install
npm run start:dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

See `.env.example` files in each directory.
